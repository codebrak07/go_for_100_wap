# 📚 Go For 100 – WAP Learning Platform

An interactive **Web Application Programming (WAP) learning platform** designed to help students practice JavaScript concepts, solve coding problems, and prepare for exams through a structured learning experience.

🌐 **Live Platform:** https://go-for-100-wap.vercel.app/

---

# 🚀 Overview

**Go For 100 – WAP** is a browser-based learning platform built to help students prepare for **WAP exams and improve JavaScript problem-solving skills**.

The platform combines **topic-wise practice, coding problems, exam simulation, AI assistance, and gamification features** to create an engaging learning experience.

Learners can practice questions, track progress, maintain streaks, unlock achievements, and even ask doubts to an integrated AI chatbot.

---

# 🤖 WAP EXPERT – AI Chatbot

The platform includes an AI assistant powered by the **Groq API** called **WAP EXPERT**.

Features:

- 💬 Ask doubts about **WAP concepts or coding problems**
- ⚡ Fast responses powered by **Groq AI**
- 🧠 Helps explain concepts and guide problem solving
- 🖼️ You can **change the AI avatar** from the **Settings panel** by pasting a custom image URL

---

# 🎯 Platform Features

### 📖 Structured Topic Practice
For **every topic**, the platform provides:

- **3 MCQs**
- **2 Integer-Type Questions**
- **5 Coding Problems**

This ensures both **concept clarity and practical coding practice**.

---

### 📝 Final Exam Simulator

A realistic **90-minute mock test** that simulates the actual WAP exam format.

**Exam Structure (100 Marks):**

- 20 MCQs
- 10 Output-based Questions
- 14 Coding Problems

---

### 🎯 Simulator Focus

We hand-picked **14 tricky and high-probability coding questions (from set 31–44)** specifically for the simulator.

The rest of the questions cover the **entire WAP syllabus**, ensuring a **complete and balanced mock exam experience**.

This helps students:

- Identify weak areas
- Practice time management
- Build exam confidence

---

# 🏆 Gamification Features

To keep learning engaging, the platform includes:

- 🔥 **Streak Tracking** – maintain daily learning streaks  
- 🏆 **Achievements System** – unlock achievements as you progress  
- 📊 Progress tracking through topic completion  

These features help build **consistent learning habits**.

---

# 🎨 Customization Features
